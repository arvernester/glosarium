Kata (Glosarium)

Memungkinkan Anda untuk mencari beberapa glosarium sekaligus dengan satu kata kunci. Sebagai informasi tambahan, jumlah permintaan dalam satu menit dibatasi hanya 10 (sepuluh) kali. Hal ini berguna untuk mengurangi beban server. Maka ada baiknya Anda menggunakan tembolok atau menyimpannya dalam pangkalan data lokal.

Pencarian Kata

Daftar atribute dan jenis data yang dapat dikirimkan ke server Glosarium.

Atribut Jenis Data Validasi Contoh Data Keterangan
GET http://api.glosarium.web.id
keyword string Harus diisi software Pencarian dapat menggunakan kata asing atau kata terjemahan.
limit integer Opsional 10 Diisi jika ingin membatasi kata dalam jumlah tertentu.
category string Opsional teknologi-informasi Filter glosarium hanya pada kategori tertentu. Lihat daftar kategori pada bagian katgeri.

Contoh kode yang dapat diimplementasikan apda PHP. Pastikan pustaka CURL sudah terinstal dalam PHP sebelum melakukan permintaan data.
